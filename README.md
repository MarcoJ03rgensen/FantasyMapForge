# FantasyMapForge

A powerful 3D random map generator for creating fantasy worlds with stunning terrain visualization. Generate procedural landscapes with mountains, valleys, rivers, and biomes, with support for both 2D and 3D exports.

## Features

- **Procedural Terrain Generation**: Create unique fantasy landscapes with customizable parameters
- **3D Visualization**: Real-time 3D rendering with interactive camera controls
- **Export Options**: Export your maps as 2D images or 3D models
- **Biome System**: Automatic biome generation based on elevation and moisture
- **Customizable Parameters**: Fine-tune terrain features including:
  - Terrain scale and roughness
  - Water levels and distribution
  - Mountain height and frequency
  - Color schemes and styling

## Getting Started

1. Open `index.html` in a modern web browser
2. Adjust the terrain parameters in the control panel
3. Click "Generate Map" to create your fantasy world
4. Use mouse controls to navigate the 3D view
5. Export your creation when satisfied
